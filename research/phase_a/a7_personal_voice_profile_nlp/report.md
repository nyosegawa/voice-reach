## 2. 背景・動機

VoiceReachのPVP（パーソナルボイスプロファイル）は、患者のメール・LINE・SNS・音声等から~2000トークンの構造化プロファイルを抽出し、LLMのシステムプロンプトに埋め込む。5ラウンドの逐次抽出（語彙→思考パターン→対人スタイル→話題傾向→回避パターン）で構成。このアプローチの学術的裏付けが必要。

## 3. 調査方法

- **検索キーワード（英語）**: author profiling, computational stylometry, writing style extraction, personality detection text, persona extraction dialogue, linguistic fingerprint
- **検索キーワード（日本語）**: 著者推定, 文体解析, 言語指紋, パーソナライズ対話, 日本語スタイル変換
- **調査対象ツール**: Stylo (R), LIWC, spaCy + GiNZA, PAN shared tasks (CLEF)
